AliceLex——lex的Javascript版本

作为编译原理课程的自学项目，目前已经实现了传统lex（典型的是GNU的flex软件）的主要功能。
已经用在了基于Html5的canvas元素的代码高亮编辑器（像notpad++）——Daisy Editor( http://yuhanghome.net/daisy-editor )中，
为其提供各种语言的词法分析。

本项目已经做为本人的毕业设计，正在进一步完善中，演示推广页面也正在编写中。
