\section{封面}
封面的例子请参看 \verb|cover.tex| 。里面的命令非常简单，一看即会。
姓名、论文题目、导师姓名职称等信息在 \verb|name.tex| 中，自己填一下。

\section{插图}
插图命令：
\begin{verbatim}
\begin{figure}[http]
\centerline{\includegraphics[width=1.97cm]{logo}}
\caption{南京大学校徽}
\label{fig:logo}
\end{figure}
\end{verbatim}

\begin{figure}[htbp]
\centerline{\includegraphics[width=1.97cm]{logo}}
\caption{南京大学校徽}
\label{fig:logo}
\end{figure}

\begin{figure}[htbp]
\centerline{\includegraphics[height=2cm]{name}}
\caption{南京大学毛体校名}
\label{fig:name}
\end{figure}

\section{编译方法}
采用 \texttt{xelatex} 编译，流程：
\begin{verbatim}
xelatex main
bibtex  main
xelatex main
xelatex main
\end{verbatim}
对于 Linux 用户，我写了个简单的 \texttt{Makefile}，在当前目录终端下输入 \texttt{make} 即可。
